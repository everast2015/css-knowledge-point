# CSS 总结

## 块级元素的特点

- 总是从新行开始
- 高度，行高，外边距和内边距都可以控制
- 宽度默认是容器的100%
- 可以容纳内联元素的其他块元素

## 行内元素的特点

行内元素不占有独立的区域，仅仅靠自身的字体的大小和图像尺寸来支持结构，一般不可以设置宽度、高度、对齐等属性，常用于控制页面中文本的样式。

常见的行内元素有：

- `<a></a>`
- `<strong></strong>`
- `<b></b>`
- `<em></em>`
- `<span></span>`

等等


- 和相邻行内元素在一行上。
- 宽高无效，但水平方向的`padding` 和 `margin` 可以设置，垂直方向的无效。
